### Процесс
- Делалось достаточно быстро, т.к. дедлайны горят везде. Много стандартных вещей брал в инете. Некое подобие бэкенда (1.5 энтити, бдшка и сопутствующее) написал очень быстро, + с воркшопа часть уже была
